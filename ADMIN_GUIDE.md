# 校园快递代领平台 - 后台管理系统使用指南

## 系统概述

本后台管理系统为校园快递代领平台提供全面的管理功能，包括用户管理、订单管理、数据统计和系统设置等模块。

## 快速开始

### 1. 管理员登录

**登录地址：** `http://localhost:5173/admin/login`

**默认管理员账号：**
- 用户名：`admin`
- 密码：`admin123`

### 2. 主要功能模块

#### 2.1 仪表板 (Dashboard)
- **功能：** 系统概览和数据统计
- **包含内容：**
  - 总用户数、总订单数、总收入、活跃用户等关键指标
  - 订单趋势图表（待集成）
  - 用户分布图表（待集成）
  - 最近活动记录

#### 2.2 用户管理 (User Management)
- **功能：** 管理系统所有注册用户
- **主要操作：**
  - 查看用户列表（用户名、邮箱、手机号、状态等）
  - 搜索用户（支持用户名、邮箱、手机号搜索）
  - 查看用户详情（包括订单统计）
  - 封禁/解封用户
  - 分页浏览用户列表

#### 2.3 订单管理 (Order Management)
- **功能：** 管理系统所有订单
- **主要操作：**
  - 查看订单列表（订单号、快递单号、发布者、接单者等）
  - 按状态筛选订单（待接单、已接单、配送中、已完成、已取消）
  - 按日期范围筛选订单
  - 查看订单详情
  - 修改订单状态
  - 分页浏览订单列表

#### 2.4 系统设置 (System Settings)
- **功能：** 管理系统基本配置
- **包含设置：**
  - 基本设置（平台名称、描述、客服信息）
  - 订单设置（自动取消时间、自动完成时间、最大订单金额）
  - 通知设置（新订单通知、完成通知、系统公告）
  - 系统维护（清除缓存、导出数据、系统重启）

## 详细使用说明

### 用户管理操作指南

#### 搜索用户
1. 在用户管理页面顶部的搜索框中输入关键词
2. 支持搜索用户名、邮箱或手机号
3. 系统会实时筛选匹配的用户

#### 查看用户详情
1. 在用户列表中找到目标用户
2. 点击"查看"按钮
3. 在弹出的对话框中查看用户详细信息

#### 封禁/解封用户
1. 在用户列表中找到目标用户
2. 点击"封禁"或"解封"按钮
3. 确认操作后，用户状态将立即更新

### 订单管理操作指南

#### 筛选订单
1. 使用状态筛选器选择订单状态
2. 使用日期选择器选择时间范围
3. 使用搜索框搜索特定订单

#### 修改订单状态
1. 在订单列表中找到目标订单
2. 点击"状态"按钮
3. 在弹出的对话框中选择新状态
4. 可添加状态变更备注（可选）
5. 确认修改后，订单状态将更新

### 系统设置操作指南

#### 修改基本设置
1. 在系统设置页面修改平台信息
2. 点击"保存设置"按钮
3. 系统会提示保存成功

#### 系统维护操作
- **清除缓存：** 清理系统临时数据
- **导出数据：** 导出用户和订单数据
- **系统重启：** 重启后台服务（谨慎使用）

## 权限管理

### 管理员权限
- 当前系统使用简单的本地存储权限验证
- 登录成功后，权限信息保存在浏览器本地存储中
- 退出登录后权限自动清除

### 路由守卫
- 所有管理员路由都受到保护
- 未登录用户访问管理员页面会自动跳转到登录页
- 登录状态过期后需要重新登录

## 技术架构

### 前端技术栈
- **框架：** Vue 3 + TypeScript
- **UI组件库：** Element Plus
- **状态管理：** Pinia
- **路由：** Vue Router
- **HTTP客户端：** Axios

### 后端集成
- **数据库：** Supabase (PostgreSQL)
- **认证：** 自定义管理员认证
- **API：** RESTful API

```bash

npm install

# 启动开发服务器
npm run dev

# 访问后台管理系统
http://localhost:5173/admin/login
```

- 确保所有API接口配置正确

- 配置适当的安全策略

## 故障排除

### 常见问题

#### 1. 登录失败
- 检查用户名和密码是否正确（默认：admin/admin123）
- 检查网络连接是否正常
- 查看浏览器控制台是否有错误信息

#### 2. 页面加载缓慢
- 检查网络连接
- 清除浏览器缓存
- 检查API接口响应时间

#### 3. 数据不显示
- 检查Supabase数据库连接
- 查看浏览器控制台错误信息
- 检查API接口是否正常返回数据

### 日志查看
- 打开浏览器开发者工具（F12）
- 查看Console标签页的错误信息
- 查看Network标签页的API请求状态

## 安全建议

1. **修改默认密码：** 生产环境务必修改管理员密码
2. **定期备份：** 定期备份用户数据和系统配置
3. **权限控制：** 严格控制管理员账号的分配
4. **日志监控：** 监控系统操作日志
5. **更新维护：** 定期更新系统依赖和补丁
