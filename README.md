# 校园快递代领平台

一个基于Vue 3 + TypeScript + Supabase的校园快递代领平台，支持发布订单、接单配送、在线支付等功能。

## 功能特性

- 🚀 **用户注册登录** - 安全的用户认证系统
- 📦 **订单管理** - 发布、接单、配送、完成全流程
- 💰 **在线支付** - 支持微信/支付宝扫码支付
- 👥 **角色区分** - 委托人（发布订单）和接单员（配送）
- 📱 **响应式设计** - 适配PC和移动端
- 🔒 **数据安全** - 基于Supabase的安全数据库

## 技术栈

- **前端框架**: Vue 3 + TypeScript + Vite
- **UI组件库**: Element Plus
- **状态管理**: Pinia
- **路由管理**: Vue Router
- **后端服务**: Supabase (PostgreSQL + REST API)
- **构建工具**: Vite

## 快速开始

### 环境要求

- Node.js 16.0 或更高版本
- npm 或 yarn 包管理器

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:5173 查看应用

### 构建生产版本

```bash
npm run build
```

## 使用指南

### 用户注册

1. 点击"注册"按钮
2. 填写真实信息（用户名、密码、昵称、手机号、邮箱、校区）
3. 注册成功后自动登录

### 发布订单（委托人）

1. 登录后进入首页
2. 点击"发布订单"
3. 填写快递信息（快递公司、单号、取送地点、时间等）
4. 设置代领费用
5. 提交订单

### 接单配送（接单员）

1. 在订单列表查看待接单订单
2. 点击"接单"按钮确认接单
3. 按照订单信息进行取件和配送
4. 完成配送后点击"完成配送"

### 支付流程

1. **委托人发起支付**：在订单状态为"配送中"或"待支付"时点击"发起支付"
2. **扫码支付**：显示接单员的收款二维码，委托人扫码支付
3. **确认支付**：委托人支付完成后点击"我已支付完成"
4. **接单员确认**：接单员确认收到款项后点击"确认支付完成"
5. **订单完成**：订单状态变为"已完成"

### 接单员设置收款信息

接单员需要先设置收款二维码才能接收支付：

1. 登录后进入个人中心
2. 点击"支付设置"
3. 上传微信或支付宝收款二维码
4. 设置完成后即可接收支付

## 数据库结构

项目使用Supabase作为后端服务，主要数据表包括：

- `users` - 用户信息表
- `orders` - 订单信息表
- `payments` - 支付记录表
- `user_payment_info` - 用户支付信息表

## 注意事项
